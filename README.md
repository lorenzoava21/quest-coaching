## I 5 Principi del Quest Coaching (v2.0)

1. **Il coaching viene prima del gioco.**
2. **Il cliente sceglie il livello di gioco.** (Metafora leggera / Strumenti / Full Quest)
3. **Meno e meglio.** Kit Minimo: Scheda + Clock + Fail Forward.
4. **La sicurezza non e opzionale.**
5. **Misura, itera, migliora.**
